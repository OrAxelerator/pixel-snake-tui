# Snake TUI

Snake mais dans le terminal.

![](./gameplay.png)

## Installation

```bash
git clone https://OrAxelerator/pixel-snake-tui.git
cd pixel-snake-tui
pip install -e .
```

## Lancement

```bash
snake
```

Options :

```bash
snake -apple 4
snake -no-colision
snake -apple 4 -no-colision
```

Touches :

- Fleches ou `h` `j` `k` `l` : deplacer le serpent
- `esc` : pause
- `r` : relancer
- `q` : quitter

Devrait marcher sur tous les OS, sauf peut-etre Windows.
